TeachTyping
Web
2023.Q2
TeachTyping is a typing game prototype project developed as university course final work. It uses PHP and MYSQL database to make it unique for a user. It has lessons and each lesson you type text seen on screen and if you make a mistake you can correct it, but it will know that you corrected it. When the lesson is finished it shows all information about how you did and saves it to the database.
