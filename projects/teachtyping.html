This project was developed for university course final work where we needed to develop some kind of website. We were 2 people I was mainly responsible for PHP and database interactions and some JS while the other person was mainly doing CSS styling and web design. The project is just a simple lesson list with basic statistics about each lesson completed.

Lessons can be divided into subsections to split apart different stages of learning or themes.

When signing up the user is created and the password is hashed so it doesn’t store in plain text. On login, the password is hashed and checked against the database saved string and if they are the same then the user access key is generated and saved as a cookie so the website doesn’t need to ask for login all the time.

In lesson if you make a mistake it is marked red with the help of class names and if you correct the mistake it is yellow. After the lesson, it calculates using class names assigned to characters to determine your accuracy with corrected as green(correct) and actual accuracy which is still counting corrected as mistakes.

The resulting statistics are shown at the end of the lesson right next to the words per minute counter and saved in the database. After completing the lesson and going back to the home screen your statistics are shown on the lesson card. They are retrieved from the database using username and lesson number.
